Stage Hand

Stage Hand is a personal speech trainer that uses a computer's webcam to record one's speech and provide real time statistics on the user's emotional expression, pace of speech, and hesistancy. Using custom acoustic and language model from Microsoft Azure's Custom Speech Services, the program is able to take into account the user's speech style and surrounding environment to provide accurate analytics.

Smart Box

The Smart Box is a home Internet of Things Hub that services as both a home assistant and smart home controller. The project is built with modules for the time, weather, personal calendar, local messaging. Additionally, using RF communication, the Smart Box is designed to communicate with various household sensors to collect data or remotely control home functions.

Nest Thermostat Virtual Chatbot

The Nest Thermostat Virtual Chatbot provides a brand new interface for Nest Thermostat users to control their household devices. This project combines Microsoft LUIS's natural language processing functionality with the Nest Thermostat API in order to make controlling these devices easier than ever before. The project is supported on Facebook Messenger, Skype, Slack, SMS, and a custom web application.
